# Atividade Individual: HTML Semântico e Estrutura

Este repositório contém a resolução da atividade prática focada no reconhecimento de especificações técnicas, paradigmas de linguagem de programação e organização de documentos HTML.

## Objetivo
Identificar as principais tags HTML e seus atributos, reconhecendo usos semânticos e aplicando esse conhecimento para estruturar corretamente uma página web através do preenchimento de um template.

## Conteúdo do Repositório

* **src/index.html**: Arquivo HTML estruturado contendo cabeçalho, navegação com âncoras internas, seções de conteúdo, tabelas e formulários com campos identificados (label/id).
* **docs/**: Relatório de pesquisa detalhado sobre 20 a 30 tags HTML, suas finalidades e atributos.
* **assets/img/**: Pasta destinada ao armazenamento das imagens utilizadas no projeto.

## O que foi desenvolvido

1. **Relatório de Pesquisa**: Levantamento técnico de tags (h1-h6, p, img, entre outras), descrevendo a função de cada uma, contexto de uso e pelo menos 3 atributos por tag com exemplos de valores.
2. **Estruturação Semântica**: Implementação de um arquivo HTML sem o uso de CSS ou atributos de estilo, focando na hierarquia de títulos, listas e tabelas.
3. **Acessibilidade**: Uso de atributos `alt` descritivos e controle de tamanho via atributos `width/height` nas imagens, além da vinculação correta em formulários.
